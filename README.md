## 1. Installation

You can install `greybeard` with your plugin manager:

```vim
Plug 'montlebalm/greybeard' " Vim-plug
```

```lua
use {"montlebalm/greybeard"} -- Packer
```

## 2. Usage

```vim
colorscheme greybeard
" For light version
set background=light
" For dark version
set background=dark
```
